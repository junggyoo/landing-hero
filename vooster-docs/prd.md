# 제품명
LandingHero

# 개요
LandingHero는 스타트업 마케터, 1인 창업자, 디지털 에이전시가 랜딩페이지 히어로 섹션을 “몇 분 안에” 생성하도록 돕는 AI 기반 웹 서비스입니다. 사용자 프롬프트를 받아 창의적인 애니메이션 디자인과 즉시 사용 가능한 HTML/CSS 코드를 자동으로 제공합니다.

# 문제 정의
1. 랜딩페이지 첫 화면(히어로 섹션)을 제작하는 데 디자인·코딩 시간이 너무 오래 걸린다.
2. A/B 테스트를 위해 히어로 섹션을 반복 생산하기 어렵다.

# 목표
주요 목표: 랜딩페이지 제작 시간을 대폭 단축한다(MVP 단계에서 80% 이상 절감 목표).

# 타깃 사용자
• 스타트업 마케터·PM  
• 1인 창업자/퍼스널 브랜더  
• 디지털 에이전시 디자이너·개발 팀

# 주요 사용 시나리오
1. 신규 제품·서비스의 랜딩페이지를 신속 제작.  
2. A/B 테스트용 히어로 섹션 다변수 생성.

# 핵심 기능 (MVP)
1. 프롬프트 기반 히어로 섹션 생성(필수)  
2. AI 프롬프트 자동 보완  
3. 실시간 반응형 미리보기  
4. 원-클릭 코드 복사

# 확장 기능(향후)
• 애니메이션 효과 선택 라이브러리  
• 브랜드 컬러 추출 및 자동 적용  
• 히어로 섹션 성능 분석 리포트

# 가치 지표 (Success Metrics)
• 제작 소요 시간: 기본 1시간 → 10분 이하  
• 월간 활성 사용자(MAU) 1,000명(런칭 3개월 내)  
• 코드 복사 클릭-투-프롬프트 비율 40% 이상

# 비즈니스 모델
Freemium  
• Free: 기본 프롬프트 생성 + 코드 복사  
• Pro(구독): AI 프롬프트 자동 보완, 반응형 미리보기 무제한, 고급 애니메이션 템플릿

# 출시 전략
• 개발 전략: MVP 우선 출시 → 사용자 피드백 기반 반복 개선  
• 초기 채널: Product Hunt / Hacker News 런칭  
• 베타 기간: 4주(무료)

# 일정(High-level Roadmap)
| 주차 | 마일스톤 | 내용 |
| --- | --- | --- |
| 1 | 제품 기획 고도화 | PRD, 사용자 여정, IA 확정 |

# 기술 스택(기본)
Next.js 15, Hono.js, Supabase, TypeScript, TailwindCSS, shadcn, lucide-react, @tanstack/react-query

# 제약 및 가정
• 브라우저 환경(반응형)만 지원, 모바일 네이티브 앱은 계획 외.  
• 애니메이션은 CSS/GSAP 범위 내에서 생성(GPU-heavy WebGL 제외).  
• 외부 AI API: Google Gemini 또는 OpenAI GPT.

# 리스크
• AI가 생성한 코드의 브라우저 호환성 문제  
• 템플릿 품질이 기대 이하일 때 이탈률 상승  
• API 사용량 증가로 인한 비용 폭증

# 향후 과제
• 사용자가 자신의 CMS(Webflow 등)에 원-클릭 연동  
• 커스텀 폰트, 이미지 업로드 지원
